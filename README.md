# 👓 시각장애인 안내 AI (Blind Guide)

> **눈이 잘 안 보이는 사람에게, 눈앞의 그림과 조각을 말로 설명해 주는 인공지능이에요.**

## 🤔 무엇을 만들었나요?

미술관에 갔다고 상상해 보세요. 눈이 보이지 않으면 그림 앞에 서 있어도 무슨 그림인지 알 수 없어요.

그래서 우리는 이런 걸 만들었어요:

1. **안경에 달린 카메라**가 눈앞의 작품을 찍어요 📷
2. **인공지능**이 사진을 보고 "무엇이 있는지" 알아내요 🧠
3. 그 사람에게 **딱 맞는 말**로 바꿔서 들려줘요 🔊

## ✨ 제일 특별한 점: 사람마다 다르게 설명해요

같은 빨간 꽃밭 그림이라도, 듣는 사람에 따라 설명이 달라져요.

| 누구에게? | 어떻게 설명해요? |
|---|---|
| 🌱 **태어날 때부터 안 보였던 분** | "빨갛다"는 말 대신 → *"따뜻한 기운의 꽃밭이에요. 이 꽃은 사람들을 기억하고 추모할 때 쓰는 꽃이에요"* (색을 본 적이 없으니, 온도나 느낌으로!) |
| 🌷 **예전에는 보였던 분** | *"붉은 양귀비꽃이 지평선까지 가득해요"* (색을 기억하니까, 색깔 그대로!) |

이게 이 프로젝트의 심장이에요. ❤️

## 🏗️ 어떻게 생겼나요? (폴더 안내)

이 프로젝트는 4개의 큰 조각으로 이루어져 있어요. 조각들은 가운데 있는 **백엔드(뇌)** 를 통해 서로 연결돼요.

```
그림을 등록하는 곳          그림을 듣는 곳
   🖥️ website  ──────┐  ┌────── 📱 app (+ 👓 안경)
                     ▼  ▼
                  🧠 backend (뇌)
                     │
                  🤖 llm (더 똑똑해지는 연습)
```

| 폴더 | 무엇을 하나요? |
|---|---|
| **`backend/`** | 프로젝트의 **뇌**. 사진을 받아서 → 무엇인지 알아보고 → 사람에게 맞는 설명을 만들어 돌려줘요. |
| **`website/`** | **작가님들이 쓰는 홈페이지**. 자기 작품 사진과 설명을 등록하면 뇌가 기억해요. |
| **`app/`** | **시각장애인이 쓰는 폰 앱**. 버튼을 누르면 설명이 글자와 목소리로 나와요. 나중엔 스마트 안경과 연결돼요. |
| **`llm/`** | **인공지능 훈련소**. 설명을 더 잘하도록 우리만의 AI를 연습시키는 곳이에요. |
| **`shared/`** | 모든 조각이 똑같이 쓰는 **약속(데이터 규칙)**. "작품 정보는 이렇게 생겼다"를 정해놨어요. |
| **`docs/`** | 설계도 문서들. 무엇을 어떤 순서로 만들지 적혀 있어요. |

> 🔧 **개발 과정이 궁금하다면** → [DEVELOPMENT.md](DEVELOPMENT.md) (구조 설계, 검증 방법, 코드 리뷰와 수정 내역)

## ✅ 지금까지 만든 것

- [x] **뇌(backend)가 작동해요** — 사진과 사용자 정보를 보내면, 그 사람에게 맞는 설명이 돌아와요
- [x] **같은 작품도 사람마다 다르게 설명해요** — 위에서 말한 제일 특별한 점이 진짜로 돼요! (실제 테스트 통과)
- [x] **진짜 사진도 알아봐요** — AI가 이미지를 보고 무엇인지 묘사하고, 등록된 작품 중 어느 것인지 맞혀요
- [x] **작가 홈페이지가 있어요** — 작품을 등록하면 뇌에 저장되고, 앱이 그걸 꺼내 써요
- [x] **폰 앱이 있어요** — 버튼을 누르면 설명이 나오고, 한국어 목소리로 읽어줘요
- [x] **우리만의 AI 연습 시작** — 남의 AI를 빌려 쓰는 대신, 우리 컴퓨터에서 도는 AI를 훈련시키는 준비를 마쳤어요
- [ ] 진짜 스마트 안경 연결 (다음 단계!)
- [ ] 진짜 데이터베이스 (지금은 연습용 데이터)

## 🚀 실행해 보기 (어른용 😄)

```bash
# 뇌(backend) 켜기
cd backend
pip install -r requirements.txt
# .env.example을 복사해 .env를 만들고 GEMINI_API_KEY를 넣어주세요
uvicorn main:app --host 0.0.0.0 --port 8000
```

```bash
# 작가 홈페이지 켜기
cd website
npm install && npm run dev
```

```bash
# 폰 앱 켜기
cd app
npm install && npx expo start
```

## 🛠️ 무엇으로 만들었나요?

- **뇌**: Python + FastAPI, 설명 만들기는 Google Gemini AI
- **홈페이지**: React + Vite
- **앱**: React Native (Expo) — 아이폰·안드로이드 둘 다
- **AI 훈련소**: Ollama + EXAONE (한국어 AI 모델)
